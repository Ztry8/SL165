# SL165 - Olymp
## Веб платформа для хранения, просмотра и добавления олимпиадных наград учеников лицея

### Пользование
Любой может просматривать все дипломы и использовать фильтры поиска чтобы найти нужный.
Администратор тот, у кого пароль который может использоваться для создания и удаления новых в базе данных.

### Установка
Вам потребуется [`nodejs`](https://nodejs.org/en):
```bash
git clone https://github.com/Ztry8/SL165.git
cd SL165
npm install
netlify dev
```

Пароль для создания и удаления новых записей об олимпиадах - **165**
